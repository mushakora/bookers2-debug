# このアプリケーションの概要
 Ruby on Rails でできる機能をとにかく実装してみる
 別のアプリケーション開発の際に参考にする

# アプリ名 bookers2-debug　
 本の紹介や感想を投稿できるサイト

 投稿、閲覧にはユーザー登録が必要
